Introduction and Learning Objectives

This project teaches the very basics of how to create optical illusion and use Adobe Premiere Pro to record this awesome moment. It is meant for beginners. The concept of optical Illusion is an illusion caused by the visual system percept what appears to differ from reality. Due to the arrangement of images, effect of colors, light source, angle or other variables which mislead visual effects can be seen.

Learning Objectives

  • Student will explore ways in which attention and preception related to illusion
  • Student will use their own reflections to build a peronsal idea about optical illusion
  • Students will learn how to use Adobe Premiere Pro to create, edit, and produce videos and films.
  • The students will be able to use a 3D printer to print object.

Techniques

    • Voice recording
  • Basic Raster Techniques
    • Change image size
    • Rotate Image
    • Subtract from the selection using the lasso tool
    • Apply basic adjustment layer techniques
  • Advanced Raster Techniques
    • Remove small objects
    • Replace a background with a layer mask
    • Add texture to an image
    • Remove a large object
  • Basic Vector Techniques
    • Combine Shapes
    • Cut and erase artwork
    • Add a line of text or paragraphs of text
    • Format Text
  • Advanced Vector Techniques

    • Recolor artwork
